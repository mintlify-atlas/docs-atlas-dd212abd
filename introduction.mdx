## Welcome to IOTA

IOTA is an asset-oriented programming model powered by the [Move programming language](https://move-language.github.io/move/introduction.html). Bringing the real world to Web3 with a scalable, decentralized and programmable DLT infrastructure.

Developed by the [IOTA Foundation](https://iota.org), IOTA provides developers with a powerful platform to build secure and efficient decentralized applications.

## Network endpoints

Connect to IOTA networks using these endpoints:

- **Localnet**: `http://127.0.0.1:9000` (for local development)
- **Devnet**: `https://api.devnet.iota.cafe` (for testing)
- **Devnet Faucet**: `https://faucet.devnet.iota.cafe:443/gas`

<Note>
  The SDK and RPC API follow a bi-weekly release cycle. Expect frequent updates during active development.
</Note>
